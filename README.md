# Интернет-магазин для компании «Кудрявый Магазин»

Проект представляет собой вёрстку интернет-магазина для интеграции с CMS OpenCart.

## Используемые технологии

- HTML5
- CSS3
- SCSS
- Vanilla JavaScript
- Bootstrap 5
- Swiper Slider
- Gulp

## Разработка

1. Установить зависимости:

```bash
npm i
```

2. Запустить разработку:

```bash
npm run dev
```

или

```bash
gulp
```

3. Сборка проекта:

```bash
npm run build
```

или

```bash
gulp build
```

## Итоговая вёрстка

Все файлы итоговой вёрстки находятся в директории `dist/`.
